## 231. [Power of Two](https://leetcode.com/problems/power-of-two/)

Given an integer `n`, return *`true` if it is a power of two. Otherwise, return `false`.*

An integer `n` is a power of two, if there exists an integer `x` such that <code>n == 2<sup>
x</sup></code>.

##### Examples

**Example 1:**

<pre>
<b>Input:</b> n = 1
<b>Output:</b> true
<b>Explanation:</b> <code>2<sup>0</sup> = 1</code>
</pre>

**Example 2:**

<pre>
<b>Input:</b> n = 16
<b>Output:</b> true
<b>Explanation:</b> <code>2<sup>4</sup> = 16</code>
</pre>

**Example 3:**

<pre>
<b>Input:</b> n = 3
<b>Output:</b> false
</pre>

##### Constraints

* <code>-2<sup>31</sup> <= n <= 2<sup>31</sup> - 1</code>

**Follow up:** Could you solve it without loops/recursion?
