## 504. [Base 7](https://leetcode.com/problems/base-7/)

Given an integer `num`, return *a string of its **base 7** representation*.

##### Examples

**Example 1:**

<pre>
<b>Input:</b> num = 100
<b>Output:</b> "202"
</pre>

**Example 2:**

<pre>
<b>Input:</b> num = -7
<b>Output:</b> "-10"
</pre>

##### Constraints

* <code>-10<sup>7</sup> <= num <= 10<sup>7</sup></code>
